# 🖥️ KiOS — Технический Манифест

**Версия:** 1.0  
**Статус:** Активная разработка  
**Цель:** Полноценная независимая операционная система уровня Windows XP

---

## 📌 Краткое описание

KiOS — это вертикально-интегрированная экосистема, разработанная с нуля:

- **Ядро:** 64-битное микроядро на C/ASM
- **Язык:** Собственный компилятор KiLang
- **Форматы:** Проприетарные бинарные форматы
- **Файловая система:** KiFS (KiOS File System)
- **Графика:** Нативный графический сервер

> *"Полный контроль от транзистора до синтаксиса."*

## 🔧 Технический стек

| Компонент | Технология |
|-----------|------------|
| Язык ядра | C99, ASM (x86_64) |
| Компилятор | GCC (cross-compile) |
| Ассемблер | NASM |
| Загрузчик | Limine |
| Эмулятор | QEMU (тестирование) |
| Железо | Lenovo G580 (цель) |

## 📂 Экосистема KiOS

### Собственные форматы файлов

| Расширение | Описание | Тип |
|------------|----------|-----|
| `.kfs` | KiOS File System Image | Системный |
| `.kelf` | KiOS Executable | Бинарный |
| `.klib` | KiOS Library | Библиотека |
| `.ktx` | KiOS Texture | Графика |
| `.kimg` | KiOS Image | Графика |
| `.kfont` | KiOS Font | Шрифт |
| `.kpkg` | KiOS Package | Пакет |
| `.kcfg` | KiOS Config | Конфигурация |

### Собственные протоколы

- **KiNet** — сетевой протокол (в будущем)
- **KiIPC** — межпроцессное взаимодействие
